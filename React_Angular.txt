++++++++ React.js - Angular.js +++++++++

1) Para crear una app de react.js
    > npx create-react-app my-app

2) Para inicializar la app de react
    > npm run dev

3) Para instalar Angular.js
     > sudo npm i -g @angular/cli

4) Para crear una app en angular
    > ng new my-app

5) Crear una app con vite
    > npm create vite@latest 

6) Para crear una app con vite y {React, Vanilla, Vue}
    > npm create vite@latest my-vue-app --template vue

7) Template con vite
    > vanilla, vanilla-ts, vue, vue-ts, react, react-ts, react-swc, react-swc-ts, preact, preact-ts, lit, lit-ts, svelte, svelte-ts.
